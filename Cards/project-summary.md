# 일본어 단어장 플래시카드 앱 - 프로젝트 요약

## 프로젝트 목표
기존 일본어 학습 앱(카드형 단어장)이 단어 수가 적어서(400~500개), 직접 단어를 추가하면서 사용할 수 있는 플래시카드 앱을 만든다.

## 현재 상태
- **1차 HTML 버전 완성** → `japanese-flashcard.html` (단일 파일, 브라우저에서 바로 실행 가능)
- TTS(음성) 동작 확인 완료 (안드로이드 Chrome)
- GitHub Pages 배포 예정 (index.html로 이름 바꿔서 업로드)

## 기술 스택
- **순수 HTML/CSS/JS** (단일 파일, 프레임워크 없음)
- **브라우저 내장 TTS** (speechSynthesis API, 일본어 ja-JP)
- **localStorage** (북마크/학습 데이터 저장)
- **PWA 배포 예정** (GitHub Pages → 홈화면 추가)

## 구현 완료된 기능
1. ✅ **카드 플래시 + TTS 3회 반복 후 자동 넘김** (▶ 버튼)
2. ✅ **스와이프 좌우 이동** + 이전/다음 버튼 + 키보드 방향키
3. ✅ **DAY별 단어 분류** (현재 DAY 1~5, 총 79단어)
4. ✅ **못 외운 단어 ★ 체크** → localStorage 저장
5. ✅ **가중치 랜덤 테스트** (★ 체크한 단어만, 맞으면 가중치↓ 틀리면 가중치↑)
6. ✅ 음성 배속 조절 (0.7x ~ 1.5x)
7. ✅ 뜻 숨기기/보기 토글

## 현재 단어 데이터 (DAY 1~5)
- DAY 1: 기본 인사·요일 (14개)
- DAY 2: 숫자·시간 (19개)
- DAY 3: 기본 명사 - 사람·장소 (15개)
- DAY 4: 기본 동사 (15개)
- DAY 5: 기본 형용사·부사 (16개)

## 단어 데이터 구조
```javascript
{
  DAY번호: {
    title: "카테고리 이름",
    words: [
      { h: "히라가나", r: "romaji", m: "[품사] 한국어 뜻" },
      // ...
    ]
  }
}
```
